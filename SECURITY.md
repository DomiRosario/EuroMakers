# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately by emailing:

- contact@euromakers.org

Do not open public GitHub issues for undisclosed vulnerabilities.

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage decision: within 7 calendar days
- Fix or mitigation plan: as soon as practical based on severity

## Scope

Security reports are especially valuable for:

- Secrets exposure
- Authentication or token misuse
- Workflow/CI abuse paths
- SSRF/path traversal/data tampering in moderation scripts

## Disclosure

After a fix is available, we may publish a summary advisory with impact and
remediation details.
